    3. Add bottom padding to the entry list container so the last entry is not covered by the FAB. Wrap the entries section (both the empty state and the AnimatePresence list) in a div with `pb-24` to ensure scrollable clearance below the FAB.

    4. Add the FAB button AFTER the JournalEditorOverlay (last element in the component return, before the closing div):
       ```jsx
       <button
         onClick={() => setShowNewForm(true)}
         aria-label="New journal entry"
         className="fixed bottom-6 right-6 z-40 flex items-center justify-center w-14 h-14 rounded-full bg-foreground text-background shadow-sm hover:scale-105 active:scale-95 transition-transform duration-100"
       >
         <Plus size={24} strokeWidth={1.5} />
       </button>
       ```

    Nothing Phone aesthetic notes:
    - `bg-foreground text-background` = inverted monochrome (white on black in dark mode, black on white in light mode)
    - `strokeWidth={1.5}` = thinner, more refined line weight matching Nothing's minimal aesthetic
    - No border-glow or dot-matrix effects — keep it clean and minimal per the established design language
    - `z-40` sits below overlays (z-50) so it doesn't show over the editor
    - `shadow-sm` provides subtle depth without breaking monochrome feel
  </action>
