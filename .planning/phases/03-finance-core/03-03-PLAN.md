**src/components/finance/BudgetGauge.tsx** — SVG circular progress ring:

Props: `{ spent: number; limit: number; onUpdateLimit: (newLimit: number) => Promise<void>; readOnly?: boolean; size?: number; strokeWidth?: number }`

Default size=160, strokeWidth=12:
- `radius = (size - strokeWidth) / 2`
- `circumference = 2 * Math.PI * radius`
- `progress = limit > 0 ? Math.min(spent / limit, 1) : 0`
- `dashOffset = circumference * (1 - progress)`

SVG structure:
- Background circle: `stroke="var(--border)"`, full ring
- Progress circle: `stroke="var(--foreground)"`, `strokeDasharray={circumference}`, `strokeDashoffset={dashOffset}`, `strokeLinecap="round"`, `transform="rotate(-90 cx cy)"` (start from 12 o'clock)
- Opacity thresholds on progress ring: `<50%: 0.3`, `50-80%: 0.55`, `>80%: 1.0`
- Progress ring animation on mount: CSS `transition: stroke-dashoffset 600ms cubic-bezier(0.16, 1, 0.3, 1)`

Center text:
- If limit > 0: formatted spent amount in `text-[1.333rem] font-mono tabular-nums` + "of {formatPHP(limit)}" below in `text-[0.778rem] text-muted-foreground`
- If limit === 0: "No budget set" in `text-[0.778rem] text-muted-foreground`
- If not readOnly: tap center text to edit budget limit (same inline edit pattern as BalanceHero but smaller)
