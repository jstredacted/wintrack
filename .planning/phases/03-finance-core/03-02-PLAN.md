**src/hooks/useFinance.ts** — Primary finance data hook:
- `useFinance(selectedMonth: string)` where selectedMonth is "YYYY-MM" format
- On mount / month change:
  1. Parse month string to get year/month via `getMonthYear()` from month utils
  2. Call `supabase.rpc('ensure_month_exists', { p_user_id: USER_ID, p_year, p_month })` to get/create month row
  3. Fetch monthly_income rows for that month via `supabase.from('monthly_income').select('*, income_sources(*)').eq('month_id', monthRow.id)`
  4. Call `supabase.rpc('populate_monthly_income', { p_user_id: USER_ID, p_month_id: monthRow.id })` then re-fetch if new rows were created
- Expose: `{ monthData, incomes, loading, error, updateBalance, updateBudgetLimit, toggleIncomeReceived, refetch }`
- `updateBalance(newBalance: number)`: optimistic update monthData.current_balance, then `supabase.from('months').update({ current_balance: newBalance }).eq('id', monthData.id).eq('user_id', USER_ID)`
- `updateBudgetLimit(newLimit: number)`: same pattern for budget_limit
- `toggleIncomeReceived(monthlyIncomeId: string, received: boolean, netPhp?: number, exchangeRate?: number, feeAmount?: number)`: call `supabase.rpc('apply_income_received', { p_monthly_income_id, p_received, p_net_php, p_exchange_rate, p_fee_amount })`, then refetch month data + incomes
- Follow useWins pattern: useState + useEffect + useCallback, cancel flag

Write tests first using vi.mock for supabase, then implement.
