## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding between bill row elements |
| sm | 8px | Compact bill row internal padding, gap between urgency border and content |
| md | 16px | Default element spacing, card padding, bill row vertical padding |
| lg | 24px | Section gaps (between Bills, Income, Budget sections on Finance page) |
| xl | 32px | Layout gaps, year overview column gutters |
| 2xl | 48px | Page top/bottom padding (`py-12` = 48px at 16px base, effectively ~43px at 18px base) |
| 3xl | 64px | Not used in this phase |

Exceptions: Bill rows use 16px vertical padding (md) and 16px horizontal padding (md) for compact density. Year overview MonthColumn uses 12px horizontal padding for tight 12-column fit.

## Typography

All sizes below are in rem relative to 18px base (`html { font-size: 18px }`). Pixel equivalents shown for clarity.

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 1rem (18px) | 400 (regular) | 1.5 | Bill row name, one-off income note, history modal entries |
| Label | 0.778rem (14px) | 400 (regular) | 1.4 | Section headers (uppercase tracking-widest), due date urgency text, date labels, muted helper text |
| Heading | 1.333rem (24px) | 300 (light) | 1.2 | Bill amounts, one-off income amounts, month column balances in year overview |
| Display | 2.667rem (48px) | 300 (light) | 1.1 | Balance hero (existing), year overview header balance |

Section headers pattern (established in Phase 3): `font-mono text-[0.778rem] uppercase tracking-widest text-muted-foreground`

## Color

All colors use oklch values from the existing shadcn Nova preset. Dark mode is the primary usage mode (Nothing Phone aesthetic).

| Role | Value (Dark) | Value (Light) | Usage |
|------|-------------|---------------|-------|
| Dominant (60%) | `--background` oklch(0.145 0 0) | oklch(1 0 0) | Page background, main surfaces |
| Secondary (30%) | `--card` oklch(0.205 0 0) | oklch(1 0 0) | Bill rows, income cards, history modal, year overview columns |
| Accent (10%) | `--foreground` oklch(0.985 0 0) | oklch(0.145 0 0) | See reserved-for list below |
| Destructive | `--destructive` oklch(0.704 0.191 22.216) | oklch(0.58 0.22 27) | Overdue bills, delete actions, revert confirmations |

Accent (foreground at full or partial opacity) reserved for:
- Bill "Mark Paid" toggle active state (full opacity)
- Waterfall chart paid step lines (full opacity)
- Year overview sparkline path (full opacity)
- Balance change indicator delta text (at 0.5 opacity — greyed out per CONTEXT.md)
- Year overview progress bar regular income fill (at 0.3 opacity)
- Year overview progress bar one-off income extension (at 0.15 opacity — visually distinct, lower opacity per CONTEXT.md)

Urgency border colors (bill due date — all use `--foreground` at varying opacity):
- Normal (7+ days): `opacity-20`, border-l-2
- Soon (4-6 days): `opacity-40`, border-l-2
- Urgent (1-3 days): `opacity-70`, border-l-3
- Overdue (past due): `--destructive` at full opacity, border-l-3

## Interaction Patterns

### Bill Paid Toggle
- Tap the toggle area on BillRow
- Optimistic: row immediately shows paid state (opacity-60, strikethrough on name)
- RPC `apply_bill_paid` runs atomically
- On error: revert optimistic state, show inline error text
- One-time paid bills: fade out and remove from list after 300ms delay

### Bill Add Inline
- "+ Add Bill" text button at bottom of BillsList
- Expands to single-row form: name (text), amount (number), due day (number 1-31), recurrence (select: one-time/recurring/ongoing)
- Enter submits, Escape collapses
- After submit: new bill appears in list, form collapses

### Balance History Indicator
- Appears only when balance_changes exist for current month
- Click opens BalanceHistoryModal

### Balance History Revert
- Each history entry has a delete/revert button
- Confirmation inline: "Revert will adjust balance by -P50,000. Confirm?"
- Two-button: "Revert" (destructive) / "Cancel"

### Year Overview Column Tap
- Tap any MonthColumn navigates to `/finance?month=YYYY-MM`
- Visual feedback: column background lightens on hover/active

### Waterfall Chart
- Non-interactive (read-only visualization)
- Shows for current and past months only
- Past months: all steps shown as solid (everything is finalized)
- Current month: paid = solid, unpaid = dashed
