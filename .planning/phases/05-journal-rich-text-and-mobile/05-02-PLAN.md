**3. Create SlashCommand extension** at `src/components/journal/SlashCommand.tsx`:
- Use `Extension.create({ name: 'slashCommand' })` from `@tiptap/core`
- Import `Suggestion` from `@tiptap/suggestion`
- Trigger on `/` character
- Commands array with objects: `{ title: string, command: (editor, range) => void }`
- Commands to support:
  - `/h2` or `/heading` -- `editor.chain().focus().deleteRange(range).setHeading({ level: 2 }).run()`
  - `/h3` -- `editor.chain().focus().deleteRange(range).setHeading({ level: 3 }).run()`
  - `/bullet` or `/ul` -- `editor.chain().focus().deleteRange(range).toggleBulletList().run()`
  - `/numbered` or `/ol` -- `editor.chain().focus().deleteRange(range).toggleOrderedList().run()`
  - `/bold` -- `editor.chain().focus().deleteRange(range).toggleBold().run()`
  - `/italic` -- `editor.chain().focus().deleteRange(range).toggleItalic().run()`
- Filter commands by matching query string against title
- Export the configured extension and the commands array

- Add a `parseBodyForTiptap` function:
  ```tsx
  function parseBodyForTiptap(body: string, format?: string): string {
    if (!body.trim()) return '';
    if (format === 'html' || body.startsWith('<')) return body;
    return body.split(/\n\n+/).map(p => `<p>${p.replace(/\n/g, '<br>')}</p>`).join('');
  }
  ```
- Add `initialBodyFormat` prop to `JournalEditorOverlayProps`: `initialBodyFormat?: string`
- Create editor instance with `useEditor`:
  ```tsx
  const editor = useEditor({
    extensions: [StarterKit, CharacterCount, Placeholder.configure({ placeholder: 'Write your entry...' }), SlashCommand],
    content: parseBodyForTiptap(initialBody, initialBodyFormat),
    shouldRerenderOnTransaction: false,
    onUpdate: ({ editor }) => {
      bodyRef.current = editor.getHTML();
      const wc = editor.storage.characterCount.words();
      setLiveWordCount(wc);
  ```
