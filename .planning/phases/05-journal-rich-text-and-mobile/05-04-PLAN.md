**2. DayStrip centering (MOB-06):**

In `DayStrip.tsx`, replace the current mount scroll behavior (`el.scrollLeft = el.scrollWidth`) with centering logic:

```tsx
useEffect(() => {
  const el = scrollRef.current;
  if (!el) return;
  const selectedCell = el.querySelector(`[data-date="${selectedDate}"]`) as HTMLElement;
  if (selectedCell) {
    const cellCenter = selectedCell.offsetLeft + selectedCell.offsetWidth / 2;
    el.scrollLeft = cellCenter - el.clientWidth / 2;
  } else {
    el.scrollLeft = el.scrollWidth; // fallback
  }
}, [selectedDate]);
```

Also add `data-date={dateStr}` attribute to each cell element (if not already present) so the querySelector can find the selected cell.

Run this scroll centering on both mount AND when `selectedDate` changes (the useEffect dependency array includes `selectedDate`).

**Tests:** Update DayStrip.test.tsx with a test: "centers the selected date cell in the scroll container." Mock `scrollRef.current` with `offsetLeft`, `offsetWidth`, `clientWidth` values and assert `scrollLeft` is set to center the cell.
