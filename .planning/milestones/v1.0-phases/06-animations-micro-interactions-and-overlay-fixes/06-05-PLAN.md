Pattern reference (from RESEARCH.md):
- Large/text buttons: `active:scale-[0.96] transition-transform duration-75`
- Icon-only buttons (< ~32px): `active:opacity-70 transition-opacity duration-75`
  (scale looks broken at small sizes)

IMPORTANT — transition coexistence in Tailwind v4:
Existing buttons already have `transition-colors` for hover. Adding `transition-transform duration-75`
alongside `transition-colors` is safe — Tailwind generates both transition properties separately.
For icon buttons using the opacity model, add `transition-opacity duration-75` (distinct from transition-colors).
