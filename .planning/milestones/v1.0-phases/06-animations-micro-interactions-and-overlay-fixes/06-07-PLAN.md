From src/components/history/DayDetail.jsx (WinRow component — the expand logic):
Current bare conditional (snaps):
```jsx
{!completed && note && expanded && (
  <p className="font-mono text-xs text-muted-foreground mt-2 pl-0 leading-relaxed">
    {note}
  </p>
)}
```
Replace with AnimatePresence + motion.div:
```jsx
import { AnimatePresence, motion } from 'motion/react';

<AnimatePresence>
  {!completed && note && expanded && (
    <motion.div
      initial={{ opacity: 0, height: 0 }}
      animate={{ opacity: 1, height: 'auto' }}
      exit={{ opacity: 0, height: 0 }}
      transition={{ duration: 0.15, ease: 'easeOut' }}
      style={{ overflow: 'hidden' }}
    >
      <p className="font-mono text-xs text-muted-foreground mt-2 leading-relaxed">
        {note}
      </p>
    </motion.div>
  )}
</AnimatePresence>
```
Note: `style={{ overflow: 'hidden' }}` on the motion.div is REQUIRED for height:auto animation — prevents content from flashing outside bounds during animation.

Also add active:opacity-70 to the expand/collapse chevron button:
```jsx
className="text-muted-foreground hover:text-foreground transition-colors active:opacity-70 transition-opacity duration-75"
```
